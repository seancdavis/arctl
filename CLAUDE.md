# CLAUDE.md

This is a Vite + React + TypeScript application with Tailwind CSS, configured for Netlify deployment.

## Netlify Primitives

When working with Netlify features, refer to the documentation files in the `.agents` directory for context and usage patterns:

- `.agents/netlify-blobs.md` - Netlify Blobs for key-value storage
- `.agents/netlify-creating-new-sites.md` - Creating and configuring Netlify sites
- `.agents/netlify-db.md` - Netlify database functionality
- `.agents/netlify-edge-functions.md` - Edge functions for low-latency compute
- `.agents/netlify-env-variables.md` - Environment variable management
- `.agents/netlify-forms.md` - Netlify Forms for form handling
- `.agents/netlify-image-cdn.md` - Image CDN and optimization
- `.agents/netlify-serverless.md` - Serverless functions

## Development

```bash
npm install    # Install dependencies
npm run dev    # Start development server
npm run build  # Build for production
npm run preview # Preview production build
```
